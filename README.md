# Candidate Management App

A simple Candidate Management application built using Next.js App Router.

## Features
- List candidates (Name, Email, Role)
- Add new candidates using a form
- Delete candidates
- Form validation
- Data persistence using localStorage

## Tech Stack
- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS

## Setup Instructions
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
4. Open http://localhost:3000
